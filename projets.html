🛠️ Stack Technologique

Frontend & Framework

  • Next.js 14+ : Framework React pour le rendu côté serveur (SSR) et la génération de sites statiques (SSG).
  • React : Bibliothèque JavaScript pour la création d'interfaces utilisateur réactives.
  • Tailwind CSS : Framework CSS utilitaire pour un design moderne, responsive et rapide à implémenter.
  • Lucide React : Bibliothèque d'icônes vectorielles pour une interface épurée.

Backend & Base de Données

  • Supabase : Plateforme Backend-as-a-Service (BaaS) fournissant l'infrastructure complète.
  • PostgreSQL : Système de gestion de base de données relationnelle pour la persistance des données.
  • Row Level Security (RLS) : Politiques de sécurité SQL pour l'isolation des données multi-tenant.
  • Supabase Auth : Gestion sécurisée de l'authentification et des sessions utilisateurs (JWT).

Services & Intégrations

  • Brevo (ex-Sendinblue) : API/SMTP pour l'automatisation du marketing et l'envoi d'emails transactionnels.
  • TypeScript : Surcouche à JavaScript pour un typage statique, garantissant la robustesse du code.

DevOps & Hébergement

  • Vercel : Plateforme de déploiement cloud optimisée pour Next.js avec CI/CD automatique.
  • GitHub : Gestionnaire de versions pour le code source et le workflow collaboratif.
  • Variables d'environnement : Gestion sécurisée des secrets et clés API en production.

Modern Web Stack : Ce mix technologique moderne démontre votre capacité à choisir des outils performants, scalables et sécurisés, ce qui est particulièrement valorisé en Master MIAGE.

🏗️ Architecture Multi-Tenant Sécurisée

3 couches de défense garantissant l'isolation des données :

Couche 1 - Frontend : Validation des inputs et filtrage par organization_id
Couche 2 - API Middleware : Vérification JWT + validation ownership de l'organization
Couche 3 - Database RLS : Politiques PostgreSQL refusant l'accès inter-organisations

Résultat : Isolation des données garantie et scalable à 1000+ organisations sans changement de code ✓
